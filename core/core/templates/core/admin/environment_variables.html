Environment Variables Documentation

Environment variables are used to configure the application without changing code. They are loaded at application startup and can override default settings.

Common Variables:
  • DEBUG - Enable/disable debug mode
  • SECRET_KEY - Secret key for cryptographic signing
  • DATABASE_URL - Database connection string
  • EMAIL_HOST - Email server host
  • ALLOWED_HOSTS - List of allowed hostnames
Warning: Be careful when modifying environment variables, as incorrect values can cause the application to malfunction.
