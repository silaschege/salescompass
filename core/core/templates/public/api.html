Getting Started

1. Get Your API Key

Sign in to your SalesCompass account and navigate to Settings → API Keys to generate your authentication token.

2. Make Your First Request

Use your API key to authenticate requests:

curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.salescompass.com/v1/accounts

3. Explore the API

Check out our comprehensive API reference below to see all available endpoints.

API Base URL
https://api.salescompass.com/v1
Rate Limits
  • • Starter: 1,000 requests/hour
  • • Professional: 5,000 requests/hour
  • • Enterprise: Unlimited

API Reference

List Accounts

GET /v1/accounts

Retrieve a list of all accounts in your CRM.

Create Account

POST /v1/accounts

Create a new account with the provided data.

Get Account

GET /v1/accounts/{id}

Retrieve details of a specific account.

List Leads

GET /v1/leads

Retrieve all leads with optional filtering.

Create Lead

POST /v1/leads

Add a new lead to your pipeline.

List Opportunities

GET /v1/opportunities

Get all sales opportunities.

Update Opportunity

PATCH /v1/opportunities/{id}

Update opportunity stage, value, or other fields.

List Cases

GET /v1/cases

Retrieve support cases.

Create Case

POST /v1/cases

Create a new support case.
